# Amend Quiz App

An app to play quizzes and get results.

## Run Locally

Clone the project

```bash
  git clone https://github.com/darshandamre/quiz-app
```

Go to the project directory

```bash
  cd quiz-app
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
